<h1 align="center" style="font-size:2.5em;">Laravel 12 API Excel Export</h1>

## About This Project
This project demonstrates how to export data from a Laravel 12 API to Excel files. It provides a simple and efficient way to generate and download Excel exports from your API endpoints, making data sharing and reporting seamless for your applications.

## Feature
- Export data to Excel (.xlsx) format via API endpoints
- Supports exporting collections, arrays, and Eloquent models
- Customizable export columns and formatting
- Handles large datasets efficiently
